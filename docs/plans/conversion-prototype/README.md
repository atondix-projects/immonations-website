## Definition of done

The prototype is done when every selected route is navigable in German, the core routes have professionally adapted English versions, all layouts work on desktop/tablet/mobile, the valuation flow works entirely in browser state, sample property interactions are visibly non-functional, supplied content and proof are placed consistently, and the P0 QA checklist passes.

No form submission, database, CRM, listing feed, consent storage, analytics, email, authentication, or other backend behavior is part of this phase.
