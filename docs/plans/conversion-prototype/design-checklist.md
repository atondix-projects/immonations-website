# Design checklist

## Foundation

- [x] Current facelift logo is used consistently: dark on light, white on dark.
- [x] Warm white is the default canvas; charcoal is selective.
- [x] Blue is a restrained accent and remains legible.
- [x] Serif display and sans body pairing has complete required weights and licensing.
- [x] Type scale, spacing, grid, borders, radii, and motion use shared tokens.
- [x] Cards are flat and editorial, not generic rounded dashboard tiles.
- [x] Supplied architecture/property imagery leads; generic stock people do not.

## Composition

- [x] Above-the-fold promise, region, and primary CTA are immediately clear.
- [x] Homepage follows the locked narrative order.
- [x] Each section has one dominant message and one obvious reading path.
- [x] Proof is adjacent to the promise it supports.
- [x] Marketing deliverables are concrete, visual, and original in composition.
- [x] Local/personal character survives the premium presentation.
- [x] Company group is a small lower-page credibility element.
- [x] Community content does not enter the homepage scope.

## Responsive and interaction

- [x] Desktop, tablet, and mobile layouts are intentionally composed.
- [x] Navigation and locale switcher work with keyboard and touch.
- [x] Tap targets are comfortably sized.
- [x] Image crops preserve architectural subjects at all breakpoints.
- [x] Hover states have equivalent focus states.
- [x] Motion is restrained and has a reduced-motion variant.
- [x] Video is muted/controlled and has text alternatives or surrounding context.
- [x] Loading and empty states do not look broken, even for local-only interactions.

## Anti-slop review

- [x] No empty “premium/exclusive/innovative” copy without evidence.
- [x] No gradient-heavy, glassmorphic, or over-shadowed treatment.
- [x] No portal-like filter density on seller routes.
- [x] No fake counters, autoplay theater, AI prominence, or decorative dashboards.
- [x] No competitor-specific copy, layout, or signature visual is reproduced.
